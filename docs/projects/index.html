👤 Face Recognition System — InsightFace + ONNX#

Computer Vision · Edge AI · Embedded Systems

A face recognition system built using InsightFace for high-accuracy face detection and embedding generation. Explored real-world edge deployment by attempting to run the model on a Raspberry Pi using ONNX Runtime — hit hardware limitations but gained deep insight into edge AI constraints and optimization challenges.

  • High-accuracy face detection & recognition via InsightFace
  • ONNX model export for runtime-agnostic deployment
  • Raspberry Pi deployment attempted — ran into memory & compute limits
  • Learned real-world edge AI tradeoffs: model size vs inference speed vs hardware

💡 Didn’t fully work on Pi — but that’s where the real engineering lessons were.
